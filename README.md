# のぶめい日記

### この日記の趣旨
ブロックチェーン、仮想通貨、DeFi、VR、メタバースなどに関する気になった記事、自分なりのまとめをリスト化しているレポジトリになります。
個人的に気になった記事を日記としてつけているものなので、あくまで参考程度にご参照ください。

基本Twitter周りから情報を集めていますが、以下リンクを参照してもらえれば自分の興味領域については知ってもらえるかと思います。
